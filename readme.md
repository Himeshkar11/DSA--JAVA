# 📖 About

This repository documents my **Data Structures & Algorithms journey in Java** while following **Kunal Kushwaha's DSA Course**.

Instead of simply copying code, I:
- ✍️ Wrote my own notes
- 💻 Typed every piece of code myself
- 🧠 Understood the intuition behind every algorithm
- 📚 Added explanations and illustrations
- 🔥 Practiced problems consistently on LeetCode

The goal of this repository is to track my progress from **Java Basics** to **Advanced DSA**.
